Do it once
```
python3 -m venv ~/VIRTUAL_ENV/django
source ~/VIRTUAL_ENV/django/bin/activate
python -m pip install Django
```
python -m django --version
 
Start project
 
```
django-admin startproject mysite
cd mysite
python manage.py runserver 9999
```
